# Onboarding Images

This directory should contain the onboarding screen images for the TaskChain app.

## Required Images

Please add the following images to this directory:

1. **on1.png** - "Build Habits Together" screen
   - Shows people collaborating or working together
   - Icon: Group/Team icon overlay

2. **on2.png** - "Stay Consistent" screen
   - Shows progress tracking or growth
   - Icon: Chart/Analytics icon overlay

3. **on3.png** - "Keep Your Chain Alive" screen
   - Shows continuity or momentum
   - Icon: Flash/Energy icon overlay

## Image Guidelines

- **Format**: PNG (with transparency support)
- **Dimensions**: 330x260 pixels (or higher resolution with same aspect ratio)
- **Style**: Modern, clean, and aligned with the app's purple theme
- **Quality**: High quality, clear imagery

## Placeholder Images

If you don't have custom images yet, you can:

1. Use stock photos from free sources like:
   - [Unsplash](https://unsplash.com/)
   - [Pexels](https://www.pexels.com/)
   - [Pixabay](https://pixabay.com/)

2. Create simple gradient backgrounds with icons using design tools

3. Use AI image generators like:
   - DALL-E
   - Midjourney
   - Stable Diffusion
