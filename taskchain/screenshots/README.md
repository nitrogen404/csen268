# Screenshots Directory

This directory contains screenshots of the TaskChain application.

## Required Screenshots

Please add the following screenshots to this directory:

1. **login.png** - Login/Sign in page
2. **home.png** - Home page showing dashboard with active chains, stats, and achievements
3. **profile.png** - Profile page with user statistics and chain information
4. **edit_profile.png** - Edit profile page with form fields
5. **settings.png** - Settings page showing notifications, preferences, and account options
6. **create_step1.png** - First step of the chain creation flow
7. **create_step2.png** - Second step of the chain creation flow

## Screenshot Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: 1080x1920 (mobile) or appropriate resolution for web
- **Quality**: High quality, clear and readable text
- **Content**: Show the actual app interface with sample data
- **Naming**: Use lowercase with underscores (e.g., `home.png`, `edit_profile.png`)

## How to Capture Screenshots

### For Web (Chrome):
1. Run the app: `flutter run -d chrome`
2. Open Chrome DevTools (F12)
3. Toggle device toolbar (Ctrl+Shift+M)
4. Select a mobile device (e.g., iPhone 12 Pro)
5. Navigate to each page
6. Use browser screenshot tools or press Ctrl+Shift+P and type "screenshot"

### For Mobile:
1. Run the app on your device/emulator
2. Navigate to each page
3. Take screenshots using device screenshot function
4. Transfer to this directory

### For macOS:
1. Run the app: `flutter run -d macos`
2. Use Cmd+Shift+4 to capture screenshots
3. Save to this directory

Once screenshots are added, they will automatically appear in the main README.md file.
